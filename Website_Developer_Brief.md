# Website Developer Brief
## Body Care E-Commerce Website
**Inspired by: saltandstone.com**
**Products: Body Wash | Body Lotion | Deluxe Body Duo Set**
**Category Structure: Shop by Scent**

---

## BRAND & DESIGN DIRECTION

- Clean, minimal, premium aesthetic with nature-inspired undertones (sea, mountains, botanicals)
- Gender-neutral, lifestyle-driven visual identity
- Muted earthy tones (sand, stone, sage, cream, warm white) with deep contrasting text
- Typography: elegant serif for headings, clean sans-serif for body copy
- High-resolution lifestyle photography — people in motion, nature settings, product close-ups
- Mobile-first responsive design across all breakpoints (mobile, tablet, desktop)
- Smooth scroll behavior and subtle micro-animations throughout

---

## 1. ANNOUNCEMENT BAR (Top of Page — All Pages)

- Sticky thin banner sitting above the navigation
- Rotating or static promotional messages, examples:
  - "Free shipping on orders $50+"
  - "Free Mini Body Wash on orders $70+"
  - "New: Take the scent quiz →"
- Text is centered, minimal, with an optional close/dismiss button
- Clickable — links to relevant offer or collection page
- Background: dark (charcoal/black) with light text OR brand accent color

---

## 2. NAVIGATION HEADER

- Fixed/sticky header that remains visible while scrolling
- Logo centered or left-aligned
- Navigation links (desktop):

  - By Scent (links to scent collection page)
  - Sets & Bundles (links to Deluxe Body Duo Set page)
  - About
  - Sustainability
- Right-side icons:
  - Search (magnifying glass icon — opens search overlay/modal)
  - Account (person icon — login/register)
  - Cart (bag/cart icon with item count badge)
- Mobile: Hamburger menu icon — full-screen or slide-in drawer navigation
- Navigation background: white or translucent with a subtle blur (glass-morphism) on scroll

### Shop Dropdown (Mega Menu):
- Body Wash
- Body Lotion
- Deluxe Body Duo Set
- Shop by Scent

---

## 3. HERO SECTION (Homepage)

- Full-width, full-viewport-height banner
- Large, cinematic lifestyle image or autoplay muted video (nature/active lifestyle)
- Headline: short, punchy brand statement (e.g., "For a Life Lived in Motion")
- Subheadline: 1–2 lines reinforcing brand values (e.g., "High-performance body care. Iconic scents. Active formulas.")
- Primary CTA button: "Shop Now" → links to All Products or By Scent page
- Secondary CTA (optional): "Take the Scent Quiz" → links to quiz page
- Text overlaid on image with sufficient contrast (dark overlay or light text zone)
- Parallax scroll effect on the hero image (subtle depth on scroll)

---

## 4. SHOP BY SCENT SECTION (Homepage + Dedicated Page)

### Homepage Teaser:
- Section heading: "Find Your Scent"
- Horizontal scrolling row or 4–5 column grid of scent cards
- Each scent card contains:
  - Scent name (e.g., "Santal & Vetiver", "Bergamot & Hinoki", "Saffron & Cedar", "Black Rose & Oud", "Neroli & Basil")
  - A short evocative descriptor (e.g., "Warm. Woody. Grounded.")
  - Top / Heart / Base fragrance notes listed
  - Associated color palette or background tone unique to that scent
  - CTA: "Explore Scent" → links to that scent's filtered collection page

### Dedicated "By Scent" Collection Page:
- Page heading: "Shop by Scent"
- Filter tabs or pill buttons at the top — one per scent
- Clicking a scent filter shows only the 3 products (Body Wash, Body Lotion, Deluxe Body Duo Set) in that scent
- Each scent category has:
  - A short atmospheric description of the scent story
  - Fragrance pyramid: Top / Heart / Base notes
  - Filtered product grid below

- Section heading: "Our Essentials" or "The Collection"
- Grid layout: 3 product cards per row on desktop, 2 on tablet, 1 on mobile
- Products shown:
  - Body Wash
  - Body Lotion

  - Product image (clean, high-res, minimal background)

  - Product name

## 6. SCENT QUIZ SECTION (Homepage + Dedicated Page)

## 14. PRODUCT DETAIL PAGE (PDP)

### Layout:
- Left: product image gallery
  - Main large image
  - Thumbnail row below (3–5 images per product)
  - Image zoom on hover
  - Lifestyle image + clean product shot mixed
- Right: product details panel

### Product Details Panel:
- Product name (large, bold)
- Short tagline / benefit statement
- Star rating + review count (clickable, jumps to reviews)
- Price (with original price + sale price if applicable)
- Scent selector: clickable pills/swatches for each available scent (e.g., Santal & Vetiver, Bergamot & Hinoki, etc.)
  - Selecting a scent updates the product image and fragrance notes dynamically
- Fragrance notes display (Top / Heart / Base) — shown after scent selection
- Quantity selector (+/- spinner)
- "Add to Cart" button (full-width, prominent)
- "Subscribe & Save" toggle (optional recurring order option at a discount %)
- Collapsible accordion sections below the add to cart button:
  - Description — product story and benefits
  - Key Ingredients — icon + name + benefit for 3–4 hero ingredients
  - Full Ingredients — complete INCI list
  - How to Use — step-by-step instructions
  - Shipping & Returns — policy summary with link to full policy
- Sticky "Add to Cart" bar appears on scroll (mobile and desktop)
- "You May Also Like" product recommendations section at bottom of PDP

## 21. FUNCTIONAL REQUIREMENTS

### E-Commerce:
- Platform: Shopify (recommended, consistent with the reference brand) or WooCommerce
- Product variants: each product has scent variants (minimum 3–5 scents per product)
- Variant inventory management per scent
- Bundle/set product support (Deluxe Body Duo Set = Body Wash + Body Lotion in same scent)
- "Sold Out" badge support per variant with optional "Notify Me" email trigger

### Performance:
- Page load target: under 2.5 seconds (Core Web Vitals compliant)
- Lazy loading for images below the fold
- Minified CSS and JS
- CDN delivery for all media assets
- WebP image format support

### SEO:
- Clean URL structure (e.g., /products/body-wash-santal-vetiver)
- Meta title and description fields editable per page and product
- Structured data markup (JSON-LD) for products (price, availability, reviews)
- Open Graph + Twitter Card meta tags for social sharing
- XML sitemap auto-generated
- 301 redirect management

### Analytics & Tracking:
- Google Analytics 4 integration
- Meta Pixel (Facebook/Instagram ads) integration
- TikTok Pixel integration
- Klaviyo or Mailchimp email marketing integration
- Hotjar or Microsoft Clarity for heatmaps (optional)

### Accessibility:
- WCAG 2.1 AA compliance
- Keyboard navigability throughout
- Alt text on all images
- Sufficient color contrast ratios
- ARIA labels on interactive elements

### Mobile:
- Fully responsive at 375px, 768px, 1024px, 1440px breakpoints
- Touch-friendly tap targets (minimum 44x44px)
- Swipeable product image galleries on mobile
- Collapsible navigation drawer on mobile

---

## 22. TECH STACK RECOMMENDATION

- **Platform**: Shopify 2.0 (Theme OS 2.0 compatible) — allows sections on all pages
- **Theme Base**: Custom build or heavily customized premium theme (e.g., Dawn, Prestige, or Symmetry as a base)
- **Frontend**: Liquid (Shopify templating) + vanilla JS or Alpine.js for interactivity
- **CSS**: Custom CSS or Tailwind CSS
- **Reviews**: Okendo or Yotpo app
- **Email**: Klaviyo (flows for welcome, abandoned cart, post-purchase, winback)
- **Quiz**: Octane AI or Typeform embedded
- **UGC Feed**: Foursixty or Taggshop
